# Food Report

Aplicación web simple para visualizar un timeline de tus comidas con imágenes locales.

## Instalación

### 1. Instalar dependencias

```bash
npm install
```

### 2. Agregar tus imágenes

Coloca todas tus fotos de comida en la carpeta `images/`. La aplicación las detectará automáticamente.

Formatos soportados:
- `.jpg` / `.jpeg`
- `.png`
- `.gif`
- `.webp`
- `.bmp`

## Uso

### Iniciar el servidor

Debes tener instalado npm

```bash
npm start
```

El servidor se iniciará en `http://localhost:3000`
